## How to add a new post

Create a file in `content/writing/` with the naming convention:

```
your-slug.en.mdx   ← English
your-slug.tr.mdx   ← Turkish (optional, falls back to EN)
```

The frontmatter fields:

```yaml
---
title: Your Post Title
date: 2026-06-12
summary: One sentence about the post.
tags:
  - tag1
  - tag2
---
```

Then write your content in MDX below the frontmatter.
